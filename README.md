# Enumerable_methods
Ruby's Enumerable Methods Project

In this project, we have created Enumerable Methods to replace Ruby's enumerable functions:

- my_each method
- my_each_with_index method
- my_select method
- my_all? method
- my_any? method
- my_none? method
- my_count method
- my_map method
- my_inject method
- multiply_els method
